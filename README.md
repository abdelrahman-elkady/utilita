## Installation

```
$ npm i -g utilita
```

## Usage

#### `set`

Copies source object deeply & recursively, and assigns new property to copy.

```javascript
import { set } from 'utilita';

const person = { name: 'Joe' };

set(person, 'age', 22); // { name: 'Joe', age: 22 }
```
